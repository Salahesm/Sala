This Bayesian network is grounded in the Facione (1990) Delphi Report — the landmark APA expert consensus on critical thinking for educational assessment. It models CT as six core cognitive skills (Interpretation, Analysis, Evaluation, Inference, Explanation, Self-Regulation), each decomposed into empirically-validated sub-skills, fed by affective dispositions, and producing observable task outcomes that aggregate into an overall CT proficiency score.
Tip: Try setting "Truth-seeking" disposition to High, then observe how belief propagates down through Analysis and Inference to the task outcomes.
